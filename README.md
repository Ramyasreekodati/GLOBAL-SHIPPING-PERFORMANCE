# Global Shipping Performance Dashboard (Power BI)

## Project Overview
The **Global Shipping Performance Dashboard** is a Business Intelligence project built using **Microsoft Power BI** to analyze and visualize global shipment data.

The objective of this project is to transform raw logistics data into actionable insights that help understand shipment trends, country-level performance, and revenue distribution. The dashboard provides an interactive environment where users can explore shipping operations through visual analytics.

---

## Dashboard Preview
### Global Shipping Performance Overview ![Shipping Performance Dashboard](https://github.com/Ramyasreekodati/GLOBAL-SHIPPING-PERFORMANCE/blob/main/shipping%20perfomance%20image.jpg)                       

### Observations
- Shipment activity shows **significant variation across months**, indicating fluctuating demand.
- The **highest shipping volume occurs around June (~1.7K boxes)**.
- The **lowest shipment volume occurs around August (~0.7K boxes)**.
- This pattern suggests a **mid-year peak in logistics activity**, followed by a decline.


--- ### Product and Total Amount Analysis ![Product and Total Amount](https://github.com/Ramyasreekodati/GLOBAL-SHIPPING-PERFORMANCE/blob/main/product%20and%20total%20amount.jpg)                     

### Observations
- Revenue distribution varies significantly across countries.
- **Australia generates the highest total shipment revenue**, making it the most valuable market.
- **India and USA follow as the next major contributors** in total shipment value.
- Countries like **Canada and New Zealand contribute comparatively lower revenue**, indicating smaller market demand.

These insights help businesses identify **high-revenue markets and potential regions for expansion**.

---

## Dashboard Components

### KPI Cards
High-level metrics providing instant visibility into shipment volume, box distribution, and geographic coverage.

### Shipment Trend Analysis
A time-series visualization showing how the number of boxes shipped changes over time.

### Country Performance Analysis
Bar charts and distribution visuals comparing shipment volumes and revenue contribution across countries.

### Sales & Product Filters
Interactive slicers allow users to filter the dashboard by:
- Country
- Sales Person
- Product

This enables dynamic analysis and deeper insights into shipping performance.
